Farland History

プライバシーポリシー

最終更新日: 2026年6月17日

モバイルゲーム「Farland History」 (以下「本アプリ」) の運営者 (以下「運営者」) は、 利用者の個人情報の保護を重要な責務と認識し、 個人情報の保護に関する法律 (個人情報保護法) その他関連法令を遵守し、 本プライバシーポリシー (以下「本ポリシー」) に基づいて利用者情報を取り扱います。

1. 取得する利用者情報

本アプリでは、 以下の情報を取得します。

1.1 利用者から直接取得する情報

  • メールアドレス — 利用者が「アカウント連携」 機能を任意で利用した場合のみ取得します。 匿名のままプレイする場合は取得しません。
  • パスワード — アカウント連携時に取得し、 暗号化したハッシュとしてのみ保存します (平文では保存しません)。
  • プレイヤー名 / アイコン選択 — 利用者がアカウント登録時に入力 / 選択した名前およびアイコン。
  • チャットメッセージ — グローバルチャット / 国家チャットに利用者が投稿した文字列。

1.2 自動的に取得する情報

  • 匿名識別子 (UUID) — 初回起動時に自動生成し、 アカウント単位での識別に使用します。 端末固有情報や広告 ID とは紐付きません。
  • ゲーム進捗データ — レベル、 ステータス、 所持装備、 ゴールド、 街の支配状況、 戦闘ログ等。
  • アクセスログ — サーバーへの API リクエスト時刻、 IP アドレス、 ユーザーエージェント情報 (アクセス解析と不正対策のため)。

2. 利用目的

取得した情報は、 以下の目的のためにのみ利用します。

  1. 本アプリのゲーム機能の提供 (キャラクター育成、 戦闘、 街の制圧、 国家間関係、 チャット等)
  2. アカウント認証および機種変更時のデータ引継ぎ
  3. 不正利用 (チート、 BOT、 RMT 等) の検知と対応
  4. サービス改善のための統計解析 (個人を特定しない集計データとして利用)
  5. 利用者からのお問い合わせへの対応
  6. 重要な変更 (規約改定、 サービス終了等) の利用者への通知

3. 第三者提供

運営者は、 以下のいずれかに該当する場合を除き、 利用者の個人情報を第三者に提供しません。

  1. 利用者の同意がある場合
  2. 法令に基づく場合 (裁判所、 警察等の公的機関からの正当な要請)
  3. 人の生命、 身体または財産の保護のために必要であり、 本人の同意を得ることが困難なとき
  4. 事業承継 (合併、 会社分割、 事業譲渡等) に伴い個人情報が承継される場合

4. 業務委託先 (個人情報の取扱いの委託)

運営者は、 サービス提供のため以下の事業者に個人情報の取扱いを委託しています。

  • Supabase Inc. — データベースおよび認証基盤の提供。 利用者情報 (メールアドレス、 パスワードハッシュ、 ゲームデータ等) はこのサービス上に保存されます。 Supabase プライバシーポリシー
  • Apple Inc. (App Store) — iOS 版の配信、 アプリ内課金処理 (将来予定)。 Apple プライバシーポリシー
  • Google LLC (Google Play) — Android 版の配信、 アプリ内課金処理 (将来予定)。 Google プライバシーポリシー
将来予告: 広告 SDK (Google AdMob 等) を導入する際には、 SDK 提供事業者を本一覧に追加し、 取得情報 / 利用目的の詳細を本ポリシーに追記します。 ATT (App Tracking Transparency) によるトラッキング許可が必要となる場合は、 アプリ内ダイアログでの明示同意を経たうえで利用します。

5. 国外へのデータ移転

Supabase 社のサーバーは、 米国およびその他の国に所在する場合があります。 本アプリの利用により、 利用者情報が日本国外のサーバーに保存・処理されることがあります。 Supabase 社は GDPR、 CCPA 等の国際的なプライバシー規制への準拠を表明しています。

6. データ保管期間

  1. 利用者がアカウントを削除するまで、 または運営者が本アプリの提供を終了するまで保管します。
  2. 利用者が「設定 → アカウント → アカウントを削除」 を実行した場合、 メールアドレス、 ゲーム進捗、 プレイヤー名等の個人を特定する情報は速やかに削除されます。
  3. 不正利用に関するログ、 法令上保管義務のある情報は、 上記にかかわらず必要な期間保管する場合があります。

7. 利用者の権利

利用者は、 自己の個人情報について以下の権利を有します。

  • 開示請求 — 運営者が保有する自己の個人情報の開示を求めることができます。
  • 訂正請求 — 個人情報の内容が事実と異なる場合、 訂正を求めることができます。
  • 削除請求 — 本アプリ内「設定 → アカウント → アカウントを削除」 から、 自身のアカウントおよび関連データの削除を求めることができます。 メールでの請求も受け付けます。
  • 利用停止請求 — 自己の個人情報の利用停止 / 第三者提供停止を求めることができます。

請求は本ポリシー末尾のメールアドレスまでご連絡ください。 ご本人確認のうえ対応します。

8. cookie・類似技術

本アプリは Web ブラウザではなくネイティブアプリのため cookie は使用しません。 認証セッションの維持には Supabase Auth が発行するアクセストークンを端末内に保存します (端末外には送信されません)。

9. 児童 (16 歳未満) のプライバシー

本アプリは全年齢を対象としていますが、 16 歳未満の利用者がアカウント連携 (メールアドレス登録) を行う場合は、 必ず保護者の同意のうえで行ってください。 保護者の同意なく 16 歳未満の利用者からメールアドレスを取得したと判明した場合、 速やかに削除します。

10. セキュリティ

  • パスワードは bcrypt によりハッシュ化して保存し、 平文では保持しません。
  • 通信は HTTPS (TLS) で暗号化されます。
  • パスワードは 8 文字以上かつ英字と数字を含む必要があり、 ブルートフォース攻撃に対する一定の耐性を確保しています。
  • 運営者の管理下にない端末紛失、 パスワードの第三者への漏洩等による情報流出について、 運営者は責任を負いません。

11. 本ポリシーの変更

  1. 運営者は、 法令の変更や本アプリの機能追加に伴い、 本ポリシーを変更することがあります。
  2. 重要な変更がある場合は、 アプリ内通知またはストア配信時のリリースノート等により周知します。
  3. 変更後のポリシーは本ページに掲載された時点から効力を生じます。
